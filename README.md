# Martens Pet Cremation Services Prototype

Static multi-page brochure website prototype for Martens Pet Cremation Services.

## Stack

- Vite
- Plain HTML
- TailwindCSS via CDN for prototype styling
- Minimal vanilla JavaScript
- Shared custom styles in `styles.css`

## Pages

- `index.html`
- `about.html`
- `services.html`
- `memorial-products.html`
- `contact.html`

## Run Locally

```bash
npm install
npm run dev
```

## Notes For WordPress Conversion

HTML comments mark sections that could later become ACF fields, including announcement text, navigation, hero content, service cards, transportation fees, memorial product link, contact details, and contact form.

Client-confirmation TODO comments are included for transportation fees and service area, communal cremation wording, memorial product link, phone numbers, addresses, and the future form connection.
